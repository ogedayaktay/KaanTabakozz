# PRD: Kaan Tabaköz Clinical Psychologist Website

## 1. Project Overview
Build a premium, high-converting, and professional website for **Kaan Tabaköz**, a Clinical Psychologist based in Lüleburgaz. The site must serve as a trust-building platform and a lead generation tool via WhatsApp integration.

## 2. Technical Stack (Recommended for Agent)
- **Frontend:** Next.js (App Router) or Vite + React.
- **Styling:** Tailwind CSS (for modern, responsive UI).
- **Icons:** Lucide-React or FontAwesome.
- **Animations:** Framer Motion (for subtle, professional transitions).
- **Deployment:** Vercel or Netlify.

## 3. Visual Identity (UI/UX)
- **Theme:** Minimalist, Premium, Sophisticated.
- **Color Palette:**
  - **Primary:** `#1A2F38` (Deep Forest Green/Slate) - Conveys stability and depth.
  - **Secondary:** `#D4AF37` (Muted Gold) - Conveys value and exclusivity.
  - **Accent:** `#7E99A3` (Calm Blue-Grey) - Conveys serenity.
  - **Background:** `#F9F9F7` (Off-White/Almond) - Clean and warm.
- **Typography:**
  - **Headings:** Serif font (e.g., Playfair Display or Cormorant Garamond) for an academic and authoritative feel.
  - **Body:** Sans-serif (e.g., Inter or Montserrat) for high readability.
